# JS FULLSTACK EXAMPLE

Dockerized NodeJs + React + Mongo + Nginx

### Installation

Its requires Docker and Docker-Compose to run.

```sh
make build
```

and then:

##### For Development:

do:

```sh
$ make mongo
$ cd client && npm start
$ cd server && npm start
```

For production environments...
do:

```sh
make up
```
